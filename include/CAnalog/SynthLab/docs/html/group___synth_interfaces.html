Detailed Description

These are pure abstrace interfaces to be used as base classes for the synth objects. These allow container objects to hold (shared) simple interface pointers, or the actual object pointers; all object interfacing is done through the common objec model interfaces here.

  • Most of these are pure abstract with functions only, while a few are more complex.
  • Some of them require special data structures or enumerations; when this happens these items will be defined above the objects, rather than in the gereralized synthstructures.h and synthconstants.h files.
  • This is just to make it easier to find them.
